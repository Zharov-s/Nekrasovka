# ABCentrum Landing

Статический лендинг промышленного парка ABCentrum.

Проект не требует сборки и готов для публикации как обычный static site, включая GitHub Pages.

## Структура

- `index.html` — основная разметка
- `styles.css` — все стили проекта
- `script.js` — интерактивность, фильтры, модальные окна, каталог лотов
- `assets/` — изображения, SVG, планировки
- `icons/` — иконки

## Локальный запуск

Можно открыть `index.html` напрямую в браузере, но для проверки поведения после публикации лучше запускать через локальный сервер.

Пример:

```bash
python3 -m http.server 8000
```

После этого откройте `http://localhost:8000/`.

## Публикация на GitHub Pages

1. Создайте новый репозиторий на GitHub.
2. Загрузите в него содержимое этой папки.
3. В настройках репозитория откройте `Settings -> Pages`.
4. В разделе `Build and deployment` выберите:
   - `Source: Deploy from a branch`
   - `Branch: main`
   - `Folder: / (root)`
5. Сохраните настройки и дождитесь публикации.

Сайт не использует сборщик, поэтому GitHub Pages должен отдавать его напрямую из корня репозитория.
